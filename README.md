# Description

No-code form builder frontend part

Backend part: https://github.com/SojebSikder/sojebform-backend

## Installation

Install all dependencies

```
yarn install
```

## Setup

Copy `example.env` to `.env` and config according to your needs.

## Running:

```bash
# development
yarn dev

# production mode
yarn start
```

For docker:

```
docker compose up
```

## Tech used

- Typescript
- Next.js
